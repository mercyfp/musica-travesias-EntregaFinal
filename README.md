# 🌿 Travesías: Memoria Sonora Amazónica

**Travesías** es una plataforma web full-stack diseñada para la preservación y difusión del paisaje auditivo y los ritmos territoriales del Caquetá. Este proyecto combina una estética visual inspirada en el movimiento **Zine** con una arquitectura de software robusta.

Desarrollado como proyecto académico para la **Universidad de la Amazonia**.

## 🛠️ Stack Tecnológico
* **Frontend:** React.js, TypeScript, Tailwind CSS, Framer Motion (Animaciones).
* **Backend:** Node.js, Express.js.
* **Base de Datos:** MongoDB (Mongoose ODM).
* **Gestión de Archivos:** Multer (Para subida de audio y portadas).
* **Iconografía:** Lucide React.

## 🚀 Funcionalidades Principales
1. **Catálogo Dinámico:** Filtrado por categorías y búsqueda en tiempo real de melodías locales.
2. **Reproductor Profesional:** Control de audio global sincronizado con el contexto de la aplicación.
3. **Sistema de Favoritos:** Persistencia de preferencias mediante LocalStorage.
4. **Módulo de Solicitudes:** Permite a la comunidad sugerir nuevos temas para el archivo.
5. **Panel de Moderación:** Interfaz administrativa para la aprobación o rechazo de solicitudes con persistencia en BD.
6. **Carga de Archivos:** Sistema de subida de archivos de audio y portadas personalizadas.

## 📦 Instalación y Configuración

### Requisitos Previos
* Node.js instalado.
* MongoDB corriendo localmente o una URI de MongoDB Atlas.

### Pasos para el Backend
1. Ir a la carpeta `backend`: `cd backend`.
2. Instalar dependencias: `npm install`.
3. Configurar variables de entorno en un archivo `.env` (PORT, MONGO_URI).
4. Iniciar servidor: `npm run dev`.

### Pasos para el Frontend
1. Ir a la carpeta raíz: `cd frontend`.
2. Instalar dependencias: `npm install`.
3. Iniciar aplicación: `npm run dev`.
